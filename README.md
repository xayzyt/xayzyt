# 你好，我是张云腾

我主要关注嵌入式系统与物联网方向，熟悉 ESP32、STM32、CH32、FreeRTOS、MQTT、LVGL，以及设备端、云端和应用端的联调流程。

## 方向

- 嵌入式 C 开发
- ESP-IDF / STM32 / CH32 / FreeRTOS
- MQTT 设备通信
- LVGL 图形界面开发
- 传感器、显示屏、控制与状态机
- Python 脚本与辅助工具

## 代表项目

- **STM32-Smart-Car**：STM32 智能小车，包含循迹、避障、跟随和蓝牙控制。
- **ESP32-S3-AIDA64-Monitor**：ESP32-S3 桌面监控屏，结合 LVGL、AIDA64 和 Wi-Fi。
- **FreeRTOS_Project**：FreeRTOS 任务、队列、同步机制等实时系统学习实践。
- **Embedded_competition**：私有竞赛项目，ESP32-P4 + CH32 + MQTT + 小程序 + 云端闭环系统。
