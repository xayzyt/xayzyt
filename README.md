## What I Build

我主要围绕 **ESP32 / STM32 / CH32V** 做嵌入式项目，关注底层驱动、RTOS 任务拆分、联网通信、屏幕 UI 和软硬件联调。

| Focus | Proof |
| --- | --- |
| Embedded C | 外设驱动、模块封装、问题定位、整机联调 |
| RTOS | FreeRTOS 任务、队列、信号量、互斥锁、事件组、状态机 |
| IoT | Wi-Fi、MQTT、HTTP、SNTP、Web 配网、HTTP SSE、cJSON |
| Display UI | LVGL、ST7789、Intel 8080 并口、DMA 刷屏 |
